Badge, Code ou Visage ?

Comparons les 3 méthodes d'accès — et voyons pourquoi le visage est la seule solution fiable pour un établissement.

🪪

Badge / Carte

Peut être volé, copié ou prêté. N'importe qui avec le badge passe.

❌
🔢

Code PIN

Peut être espionné ou partagé. Un code ne prouve pas l'identité.

❌
🧑

Le Visage

Unique et impossible à voler. Vous ÊTES votre clé d'accès.

✅

💡 L'idée : Au lieu de vérifier ce que vous possédez (badge) ou savez (code), on vérifie qui vous êtes. Votre visage ne peut pas être oublié, perdu ou prêté.

Club IA04 — Besoins du Modèle
04Besoins du Modèle

De quoi a besoin le modèle IA ?

Un modèle performant repose sur 3 piliers : données d'entraînement, puissance de calcul et algorithmes optimisés.

📸

Données d'entraînement

50 à 200 photos par personne sous différents angles, éclairages et expressions. Plus = meilleur.

🧠

Réseau de neurones (CNN)

Un réseau convolutif profond apprend à extraire les caractéristiques uniques de chaque visage.

🎯

Fonction de perte (ArcFace)

Entraîne le modèle à maximiser la distance entre les identités différentes et la minimiser pour la même personne.

✅

Validation & tests

Le modèle est testé sur des visages jamais vus pour garantir la généralisation. Objectif : 99%+ de précision.

Club IA05 — Embarqué + IA
05Système Embarqué

IA Embarquée : Le Cerveau au Portail

Le modèle IA tourne directement sur un mini-ordinateur fixé au portail — aucun cloud, aucun internet nécessaire.

🖥️

NVIDIA Jetson Orin Nano

Mini-PC avec GPU intégré — exécute le modèle IA en temps réel à 30 FPS. Taille d'une carte de crédit.

📷

Caméra IR + RGB

Capture en infrarouge + couleur. Fonctionne de jour comme de nuit, même dans l'obscurité totale.

⚡

TensorRT (Optimisation)

Le modèle est compilé et optimisé pour le GPU embarqué → inférence 5x plus rapide qu'en Python natif.

🔌

GPIO → Relais → Portail

Le Jetson envoie un signal électrique au relais qui commande l'ouverture du portail. Tout automatique.

Du Pixel à la Décision : Code Embarqué

Voici le chemin complet qu'une image parcourt dans le système embarqué, du capteur au portail.

▶
📊
4. Matching
Distance cosinus vs base
cosine_sim()
▶
🚪
5. Action
GPIO ouvre ou alerte

🛡️

Anti-spoofing

Détecte photos, masques et vidéos via analyse de texture IR + clignement.

📋

Log local

SQLite enregistre chaque passage : horodatage, ID, score, photo.

🔄

Mise à jour

Nouveau personnel ? On ajoute ses photos via USB ou réseau local sécurisé.

🐍 Stack logiciel : Python 3.10 + PyTorch + OpenCV + TensorRT + SQLite — le tout optimisé pour NVIDIA Jetson avec JetPack SDK.
